使用 HTML 构建易于理解的 UI 结构，使用 CSS 来描述 UI样式和布局，一种优雅的 Canvas 绘图实现。

实现原理：
构建虚拟DOM 树，依据 CSS 规范计算样式，使用 CSS盒模型对 DOM 进行布局，计算出所有元素的位置。最后将 DOM树通过 Canvas Api 进行绘制。
兼容性：
HTML 样式和布局计算不依赖于特定 JavaScript 宿主环境，因此可以在Node，游览器，小程序等环境上使用
